# Changelog

All notable changes to the "StarCap Dark" theme will be documented in this file.

## [1.0.0] - 2024-05-08

### Added
- Initial release of StarCap Dark theme
- Complete color palette with StarCap brand colors
- Syntax highlighting for TypeScript, JavaScript, React, HTML, CSS
- Custom terminal colors
- Git decorations
- Semantic colors for success, error, warning states

### Colors
- Blaze Dark (#0F1923) - Editor background
- Blaze Black (#05090C) - Status bar, title bar
- Blaze Card (#1A242D) - Sidebar, panels
- Neon Green (#00E676) - Functions, success states
- Neon Red (#FF003C) - Tags, strings, errors
- Neon Blue (#2979FF) - Keywords, highlights
- Text White (#FFFFFF) - Primary text
- Text Gray (#9EA6AD) - Secondary text, comments
